# Вариант 4

Отсортировать массив целых чисел при помощи TimSort

# Трассировка
Для анализа работы программы используйте `strace`. Флаг `-f` обязателен для отслеживания порожденных потоков.

```bash
strace -f -o trace.log ./timsort 1000 4
```
-f — отслеживать дочерние потоки (созданные через clone).

-o trace.log — сохранить результат в файл для анализа.

# Основные системные вызовы в проекте

clone3 / clone: создание новых потоков исполнения.

futex: реализация блокировок семафора и ожидание завершения потоков (pthread_join).

mmap / mprotect: выделение памяти под стек потоков и защиту данных.

write: вывод состояния сортировки в терминал.
